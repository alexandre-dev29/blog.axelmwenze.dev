### Running the project

#### this project was configure to run with  but you can run it with any graphql server that you want
- in `apps/front/` create a .env file using the .env.example structure
- in `graphql/Schemas` create your schemas that matches your graphql api
- Run
- `yarn install` or
- `npm install`
- generate types from Graphql Schemas
- ``yarn run codegen``
- and then
- ``yarn run start``
